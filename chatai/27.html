❓:(2020 秋・沈阳期末）某单位共有 500 名职工, 其中不到 35 岁的有 125 人, 35-49 岁的有 a 人, 50 岁及以上的有 b 人, 现用分层抽样的方法, 从中抽出 100 名职工了解他们的健康情况. 如果已知 35-49 岁的职工抽取了 56 人, 则 50 岁及以上的职工抽取的人数为 ( )A. 19 B. 95 C. 220 D. 280

🧠:好，我现在要解决这个统计题目。题目是说，某单位共有500名职工，分成三个年龄段：不到35岁的有125人，35-49岁的有a人，50岁及以上的有b人。现在用分层抽样的方法抽取100名职工，其中35-49岁的被抽取了56人。问题是要找出50岁及以上的职工被抽取的人数是多少，选项是A.19，B.95，C.220，D.280。首先，我需要回忆分层抽样的基本概念。分层抽样是将总体分成不同的层，然后按比例从每一层中抽取样本。也就是说，每个层抽取的样本数量应该与该层在总体中的比例相等。题目里说总共有500人，分成三个层：不到35岁的125人，35-49岁的a人，50岁及以上的b人。所以首先应该满足总人数是这三个部分的和，也就是125 + a + b = 500。这里可能需要用到这个等式来解出a和b的值或者他们的比例关系。接下来题目说用分层抽样抽取了100人，其中35-49岁的抽取了56人。问题是要找50岁及以上的被抽取的人数。那么，首先应该确定每个层在总体中的比例，然后根据这个比例来计算每个层应该抽取多少人。首先，我需要算出各个层在总体中的比例。对于35-49岁这个层来说，人数是a，占总体的比例是a/500。根据分层抽样的方法，抽取的样本数应该是总样本数100乘以这个比例，也就是100*(a/500) = a/5。而题目中给出这个层被抽取了56人，所以可以列出方程：a/5 = 56，解得a = 56*5 = 280。所以a的值是280人。接下来，根据总人数500，我们可以求出b的值。因为总人数是500，所以125（不到35岁） + 280（35-49岁） + b（50岁及以上） = 500。计算这个等式：125 + 280 = 405，所以b = 500 - 405 = 95。所以，50岁及以上的人数是95人。现在，题目要问的是50岁及以上的人被抽取了多少人。同样按照分层抽样的方法，抽取的样本数应该是总样本数100乘以该层的比例，也就是b/500。所以抽取人数是100*(95/500) = (100*95)/500。计算这个的话，95除以5等于19，所以结果是19。所以答案应该是选项A.19。不过，为了确保没错，再仔细检查一遍步骤：1. 总人数是500，已知三个层的总和为125 + a + b = 500。2. 分层抽样抽取100人，其中35-49岁抽取56人。根据分层抽样的原理，抽取比例等于该层在总体中的比例。3. 35-49岁抽取人数是56，对应的比例是56/100，而该层在总体中的比例是a/500。所以等式成立：a/500 = 56/100，解得a = (56/100)*500 = 56*5=280。正确。4. 计算b的值：125 + 280 = 405，所以b=500-405=95。正确。5. 50岁及以上的抽取人数是该层比例乘以总样本数：b/500 *100 = 95/500 *100 = (95*100)/500 = 95/5=19。正确。所以选项A.19是正确的。不过，这里可能需要注意的是是否分层抽样的比例确实是按各层人数的比例来分配的。比如，每个层中抽取的样本数等于该层人数占总体的比例乘以总样本量。这确实是分层抽样的基本方法，所以没错。再确认一下计算过程，特别是分数的计算是否有误。比如，95除以500等于0.19，再乘以100得到19，所以没错。选项A是19，正确。可能选项中还有B选项是95，但95是50岁及以上的职工总数，而不是样本数，所以这里需要看清楚题目问的是抽取的人数，即样本中的数量，所以正确选项应该是A.19，而B选项是总体中的数量，不是答案。综上，正确答案是选项A.19，应选A。
